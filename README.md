アプリ名
movie-review

サイト概要

サイトテーマ
映画の感想を投稿するサイト

テーマを選んだ理由
自分が映画を見るのが好きなんですが、
みておしまいとなってしまうことが多くて、あとで内容を思い出せなかったりするので
映画をみた感想や評価をつけれて他のユーザーと共有できたらと思ったから。


ターゲットユーザ
映画好きで、レビューを書いてみたい人、また他の人の感想を知りたい人。
映画の感想を参考に見るものを決めたい人。

主な利用シーン
映画をみたあと感想を書く時
見たい映画を探す時
